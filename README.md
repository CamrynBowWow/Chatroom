# Chatroom

## How To Create Database

1. Open up the 'Package Manager Console'
2. Change <b>Default project</b> to 'Chatroom.Plugins.EFCore'
3. Then type or paste in 'Update-Database Init -Context ChatroomContext' and press enter.
